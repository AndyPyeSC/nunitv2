NUnit 3.0 Test Adapter for Visual Studio - CTP 1 - In Process

Features

  • Initial preview release of the test adapter using NUnit 3.0. Note that the adapter may not be used to run tests written against earlier versions of NUnit. The original adapter is still available for that purpose and both adapters may be installed if necessary.
